# 1/25 IT Infra - State

### 상태 저장/비저장 (Stateful/Stateless)

- 상태저장
    - 상태 전이를 이식해서 처리하며 과거의 경위를 이해한 상태에서 응답을 준다
    - 복잡한 처리가 가능하지만, 시스템의 복잡성이 커진다
    - 대표적으로 CPU의 프로세스 처리시 활용된다
- 상태비저장
    - 매번 신규 요청이 생성된다
    - 간단하며, 성능이나 안정성 측면에서 우수하다
    - 대표적으로 HTTP에서 사용된다 (Session을 활용해 Stateful 하게 사용하기도 한다)
